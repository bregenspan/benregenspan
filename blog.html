Open-source Projects

Django Ajax Changelist (built)

This small Django app lets you specify fields in your Django admin's object list that you would like to be AJAX-editable. This is intended for when users of the admin need to update certain properties on a number of items, and don't want to have to click through to the object detail page, then return back to the change list each time.

Jewnicorn

This is a lightly modified fork of gunicorn, the widely-used all-Python HTTP server. It preserves all of gunicorn's features, but prevents it from serving content on Shabbos. ("astral" is used for finding local sun-up/sun-down times).

While this project exists entirely as a dumb joke, it should be noted that some sites do close down on Shabbos, B&H likely being the most prominent example. There is an interesting discussion of this subject over on StackExchange.

Fight For Your Right (To Pay With a Credit Card) (javascript, chrome-extension)

This simple Chrome extension defaults Paypal's payment option to credit card.

One Plus Forward, Two Plus Backward

This Chrome extension adds a limit of 140 characters to Google Plus and hides the buttons for doing fancy things that wouldn't be possible on Twitter, like recording video. In the interests of exceptional yet subtle UI, the HTML5 Audio API (via Buzz!) is used to play an airhorn effect when the user exceeds the character limit, and an image of the Dalai Lama flashes on the screen to remind the user of the beauty that can be found in simplicity. (Sadly, some updates are required to make this work again after the latest Google+ redesign)

Live Twitter Background

This is an old project and there are a number of things I'd clean up about the code. Nevertheless, if you'd like to use something written in PHP to periodically update your Twitter background to show a highly unecessary list of recent followers + your current Skype status, this could be a good fit.

My contributions to other open-source projects currently tend to be very small -- I'll commit documentation fixes, tiny features, etc. I hope to make more substantial open-soruce contributions in future.
